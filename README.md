## Задачи

### Задача 1 (базовая)
Вычислить площадь трапеции по основаниям a, b и высоте h.

**Формула:** S = (a + b) × h / 2

### Задача 2 (средняя)
Вычислить среднюю линию трапеции.

**Формула:** m = (a + b) / 2

### Задача 3 (расширенная)
Вычислить периметр равнобедренной трапеции, если известны основания a, b и высота h.

**Формулы:**
- Боковая сторона: c = √(h² + ((a-b)/2)²)
- Периметр: P = a + b + 2c
